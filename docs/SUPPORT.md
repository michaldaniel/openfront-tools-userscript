# Support

Support and feature requests: Discord `@morning.bird`.

When reporting a bug, include:

- OpenFront Tools version
- browser and userscript manager
- enabled panels
- what happened and what you expected
- console logs after enabling debug mode

Enable debug mode in the OpenFront page console:

```js
OpenFrontTools.enableDebug()
```

Disable it:

```js
OpenFrontTools.disableDebug()
```
